# PRD：InkMind 阅读划线内容统一管理产品（MVP 升级版）

> 本文档由 `InkMind_PRD_AI_MVP.docx` 整理为 Markdown，便于版本管理与协作。

## 一、产品定位

为阅读者打造的私人共鸣收藏馆。收藏触动你的句子，AI 解读此刻的自己。

## 二、产品背景

用户在阅读纸质书、电子书、文章时，常划线或截图保存喜欢的句子，但内容分散、难检索、缺少统一沉淀与回顾。

## 三、产品目标（MVP）

1. 快速收藏句子
2. 自动整理分类
3. 支持搜索回顾
4. AI 解读用户为何产生共鸣
5. 提升留存与分享率

## 四、目标用户

阅读爱好者、内容创作者、学生群体、关注自我成长与情绪表达的人群。

## 五、核心痛点

收藏后不再回看；内容分散；想找找不到；记录麻烦；缺少精神陪伴。

## 六、核心价值主张

把阅读时的灵感瞬间，沉淀成长期资产；通过 AI 解读文字背后的情绪与意义。

## 七、核心功能（MVP）

1. 内容库首页（列表 / 搜索 / 标签）
2. 手动输入句子
3. 拍照 OCR 识别录入
4. AI 自动标签分类
5. AI 单句解读
6. 每日随机回顾
7. 分享卡片

## 八、AI 单句解读（MVP 卖点）

用户保存句子后，AI 输出：

1. 句子核心含义
2. 为什么容易引起共鸣
3. 一个反思问题

**示例：** 你收藏这句话，可能说明你正在寻找方向感。

## 九、非 MVP（第二阶段）

阅读人格报告；100 条收藏趋势分析；情绪曲线；书籍推荐；社区广场；多语言支持；Kindle 同步。

## 十、页面结构

- **首页**：内容列表 / 搜索 / 添加
- **添加页**：OCR / 手动输入
- **详情页**：编辑 / 删除 / AI 解读 / 分享
- **个人中心**：统计 / 会员

## 十一、用户流程

看到一句话 → 收藏 → AI 解读 → 感到被理解 → 持续使用 → 回顾分享

## 十二、商业模式

| 版本 | 能力 |
|------|------|
| **免费版** | 100 条收藏、基础解读 |
| **Pro 版** | 无限收藏、深度 AI 分析、OCR 无限次、云同步、高级卡片模板 |

## 十三、关键指标

每周新增收藏数、AI 解读使用率、7 日留存、人均收藏数、分享率、付费转化率。

## 十四、技术方案

| 层级 | 选型 |
|------|------|
| 前端 | Taro 小程序，后续考虑 App |
| 后端 | Node.js + Express + MySQL |
| OCR | 待定 |
| AI | 待定 |

## 十五、开发排期（14 天）

- **第 1 周**：登录、收藏 CRUD、首页、搜索
- **第 2 周**：OCR、AI 标签、AI 解读、分享卡片、上线部署
